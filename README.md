# Bank_customer_churn_Analysis_project
End-to-end data analysis project using Excel, SQL, Python and Power BI.

## 📌 Project Overview

Customer churn is a major challenge for banks as retaining existing customers is more cost-effective than acquiring new ones.
This project analyzes customer demographics, financial behavior, and account details to identify factors influencing churn and provides business insights through interactive dashboards.

## 🎯 Objectives

- Clean and prepare raw banking data.
- Analyze customer demographics and financial attributes.
- Identify patterns associated with customer churn.
- Answer business questions using SQL.
- Perform Exploratory Data Analysis (EDA) using Python.
- Build an interactive Power BI dashboard for business users.

## 🛠️ Tools & Technologies

- **Excel** – Data Cleaning
- **SQL (MySQL)** – Data Querying & Business Analysis
- **Python**
  - Pandas

## 📊 Key Business Insights

- Customers with lower tenure show higher churn.
- Customer distribution varies significantly across different geographies.
- Financial characteristics such as balance and credit score influence churn behavior.
- Customer engagement and account activity play an important role in retention.
